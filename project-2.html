Moviegraphy

This web app allows users to search for movie titles by using an API to pull data and display the results. Users can also add films to their personal watchlist, which is saved in their local storage for easy access later on. Built with HTML, CSS and JavaScript.

Live Link

Project Overview

Spending ages trying to find something to watch? Constantly rewatching the same show? This app is perfect for you! Moviography allows you to easily search for films and add interesting titles to watchlist. Next time a friend mentions a good film, with Moviography you can quickly find out its rating, brief summary of the plot, length and genre. Looks intriguing? Add it to your watchlist!

Features

  • Search for films pulls data from API and renders the results
  • User can switch between film search and watchlist view (top-right corner)
  • User is able to add and remove films to/from the watchlist
  • Films in watchlist are saved to local storage
